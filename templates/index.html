LM4ACE: A universal deep learning architecture for bioactive peptide prediction

The webserver is the implementation of the paper UniDL4BioPep: UniDL4BioPep: A universal deep learning architecture for binary classification in peptide bioactivity



{{ prediction_text }}

Usage of the webserver:

1. Select a prediction model based on their index, 1. logistic regression (LR); 2. Support vector machine (SVM); 3. Multilayer perceptrons (MLP)
2. Input your peptide sequence and separate them by comma
Example: you can input 1 in the first box and ARG,IPI,CCL,FIRST; Than means you want to use "ACE inhibitory activity prediction model" to predict the bioactivity of the four peptide sequences
